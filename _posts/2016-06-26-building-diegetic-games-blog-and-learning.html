I’ve been working on the site for my game studio Diegetic Games and made some major progress this weekend.

I’m hosting the site on github pages and I’m now using Jekyll which makes content management super easy. Jekyll lets me use templates for different page types and includes for the head, header, and footer. It auto assembles a main blog page and I can write my posts in markdown. It was really easy to get Jekyll set up.

I’m going to start shifting some of my game writing and announcements over to the Diegetic Games blog which you can find at: Diegetic Games Blog

I’m also going to start adding learning resources for folks getting into story games and game design. The first page is up here: Learn about Story Games - it’s an intro to storytelling games, RPGs and LARPs.
